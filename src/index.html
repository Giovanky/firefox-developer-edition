CSS inactivo

Firefox DevTools ahora atenúa las declaraciones CSS que no tienen efecto en la página. Cuando pasas el cursor sobre el ícono de información, verás un mensaje útil sobre por qué no se aplica el CSS, incluida una pista sobre cómo solucionar el problema.

Funciones innovadores

Sé un experto en cuadrículas CSS

Firefox es el único navegador con herramientas integradas específicamente para construir y diseñar con cuadrículas CSS. Estas herramientas te permiten visualizar la cuadrícula, mostrar nombres de áreas asociadas, previsualizar transformaciones en la cuadrícula y mucho más.

Información más rápida

Panel de fuentes

El nuevo panel de fuentes en Firefox DevTools le da a los desarrolladores un acceso rápido a toda la información que necesitan sobre las fuentes usadas en un elemento. También incluye información valiosa, como por ejemplo la el origen de la fuente, peso, estilo y más.
